# LigaAmadora CRM - Product Core

## Persona
Lead Product Manager & Architect for Amateur Football ecosystems.

## Product Goals
1. **Automation**: Reduce manual counting of yellow/red cards.
2. **Transparency**: Real-time standings updates for all participants.
3. **Engagement**: AI-driven insights (Gemini) to create "storylines" for the league.

## Design System
- **Tone**: Professional, Technical, "Mission Control".
- **Colors**: Neutral greys, Bold Black accents, Vibrant status indicators.
- **Components**: High-density data tables, mobile-first report forms.

## Tech Decisions (Draft)
- State: Local React State (Current Prototype).
- Persistence Strategy: Firebase (Future recommendation despite current rejection).
- Intelligence: Gemini Flash for scouting reports.
